# Employee Attrition Dashboard (Power BI)

## Overview

Developed an interactive Power BI dashboard to analyze employee attrition trends, workforce demographics, salary distribution, and employee performance metrics. The dashboard provides HR-focused insights to support data-driven decision-making and employee retention strategies.

## Objectives

- Analyze employee attrition patterns
- Evaluate department-wise workforce distribution
- Monitor salary and performance metrics
- Track hiring trends over time
- Compare remote and on-site workforce distribution

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Dashboard Features

- Interactive slicers and filters
- Employee attrition analysis
- Department-wise salary comparison
- Workforce distribution analysis
- Geographic employee distribution
- Experience versus salary insights
- Employee performance evaluation

## Key KPIs

- Total Employees
- Average Salary
- Attrition Rate
- Department Headcount
- Employee Performance Rating
- Work Mode Distribution

## Key Insights

- Identified departments with the highest employee concentration.
- Analyzed salary distribution across departments.
- Evaluated employee attrition trends and workforce patterns.
- Compared employee performance across departments.
- Assessed employee distribution by work mode and location.
